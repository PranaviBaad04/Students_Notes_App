# 📚 Students Notes App

A web-based platform developed using Python Flask, HTML, and CSS that allows students to upload, browse, search, and download academic notes. The application helps students organize study materials efficiently by providing semester-wise and subject-wise filtering options.

## 🚀 Project Overview

Students Notes App is designed to simplify the sharing and access of academic notes among students. Users can upload notes related to different subjects and semesters, while other students can quickly find the required study material using search and filter functionalities.

The application provides an easy-to-use interface for managing educational resources and promotes collaborative learning.

## ✨ Features
### 📤 Upload Notes
- Upload study notes in PDF format.
- Add note details such as:
- Title
- Subject
- Semester
- Description
### 🔍 Search Notes
- Search notes by title or keywords.
- Instant filtering of available notes.
### 🎓 Semester-wise Filtering
- View notes based on selected semester.
- Helps students quickly access relevant study material.
### 📖 Subject-wise Filtering
- Filter notes according to specific subjects.
- Makes note discovery easier and more organized.
### 📥 Download Notes
- Students can download uploaded notes for offline study.
### 🖥️ Responsive User Interface
- Clean and user-friendly design.
- Works across desktop and mobile devices.
## 🛠️ Technologies Used
### Backend
- Python
- Flask
### Frontend
- HTML5
- CSS3
